Fonts Present in Mpisto OS

Some fonts may fallback to a generic or to another font for some cases such as Arial Nova font if font not found. If font is on Google Fonts or on other sources (such as FANDOM Wikis, with credits to it too) present, we'll use that instead, with fallback of course for Opera Mini and older browsers. So if your browser is not Opera Mini and do not have some of the fonts, a few of them will are imported so no fallback should be seen.
